

# A股市场温度计

查询今日A股市场温度的skill，为您的投资决策提供参考依据。

## 功能概述

A股市场温度计是一款专门用于监测A股市场温度的工具。通过读取配置文件中的指数列表，自动计算各指数的 PE(TTM) 值和对应温度，并以直观的格式呈现市场状态。

### 核心能力

- **多指数监控**：支持同时监控多个A股指数
- **PE温度计算**：自动获取指数的 PE(TTM) 数据并转换为温度值
- **状态判断**：将温度划分为低温、中温、高温三个区间，辅助判断市场热度
- **灵活配置**：支持自定义监控指数和温度阈值

## 温度状态说明

| 状态 | 温度范围 | 含义 |
|------|----------|------|
| 🟢 低温 | < 30°C | 市场估值偏低，可能存在投资机会 |
| 🟡 中温 | 30°C - 80°C | 市场估值合理，处于正常运行区间 |
| 🔴 高温 | > 80°C | 市场估值偏高，需注意风险 |

## 输出格式示例

```
📊 今日A股市场温度报告 (2026-04-21)

指数            PE      温度    状态
上证50        12.5    25°C    低温
深证100       18.3    45°C    中温
中证1000      35.2    86°C    高温
```

## 快速开始

### 环境要求

- Python 3.x
- 已安装akshare包
- 网络连接（获取实时行情数据）

### 使用方法

-将skill文件夹放置在AI agent的skills目录下，当用户询问"今日市场温度"、"今日A股温度"、"市场热度"、"大盘温度"或类似问题时触发此技能。
-可结合定时任务，每天定时推送报告

### 配置文件说明

编辑 `scripts/config.json` 来自定义监控的指数：

```json
{
  "indices": [
    {"code": "000016", "name": "上证50"},
    {"code": "000300", "name": "沪深300"},
    {"code": "000009", "name": "上证380"},
    {"code": "399005", "name": "创业板50"},
    {"code": "000905", "name": "中证500"},
    {"code": "000011", "name": "深证红利"},
    {"code": "000012", "name": "深证100"},
    {"code": "000852", "name": "中证1000"}
  ]
}
```

**配置说明**：

- `code`：指数代码（上证指数为 000001，深证成指为 399001，创业板指为 399006）
- `name`：指数显示名称


### 温度阈值说明

当前版本的温度阈值定义如下：

- 低温阈值：30°C
- 高温阈值：80°C

## 注意事项

- 程序需要网络连接以获取实时行情数据
- 温度数据仅供参考，不构成投资建议
- 市场温度低不代表一定会上涨，温度高也不代表会下跌
